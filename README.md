### Interfaz intuitiva y modo de juego contra una CPU
#### ¿Para qué se implementó?
Para ofrecer una interfaz más clara, atractiva y funcional, mejorando la experiencia del usuario al iniciar y configurar el juego. Además, se incorporó el modo contra CPU, brindando una opción de juego individual más dinámica.

#### ¿Por qué se consideró necesario?
El menú original era limitado y poco intuitivo. Se buscó optimizar la navegación, hacerlo más accesible y visual, e incluir nuevas opciones que aumentaran la interactividad y el alcance del sistema.

#### ¿Cómo se llevó a cabo su implementación?
Se rediseñó el menú en consola utilizando colores, símbolos y una estructura modular. Se añadieron opciones interactivas y mensajes guiados para mejorar la usabilidad, junto con la integración del modo CPU, que permite enfrentar al jugador con la inteligencia del sistema de forma automática.
